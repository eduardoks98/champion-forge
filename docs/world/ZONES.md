# Zonas do Mundo

## Visao Geral

O mundo de Champion Forge e dividido em 6 zonas principais, cada uma instanciada por party.

```
┌─────────────────────────────────────────────────────────────┐
│                    MAPA DO MUNDO                            │
│                                                             │
│   [1. Floresta]  ──────  [2. Cavernas]  ──────  [3. Deserto]│
│       Lv 1-10               Lv 10-20              Lv 20-30  │
│         │                      │                      │     │
│         └──────────┬───────────┴──────────┬───────────┘     │
│                    │                      │                 │
│               [4. Ruinas]           [5. Vulcao]             │
│                Lv 30-40              Lv 40-50               │
│                    │                      │                 │
│                    └──────────┬───────────┘                 │
│                               │                             │
│                         [6. Abismo]                         │
│                           Lv 50+                            │
│                          (Endgame)                          │
└─────────────────────────────────────────────────────────────┘
```

## Progressao Entre Zonas

```
Level 1  → [Floresta]    → Level 10
Level 10 → [Cavernas]    → Level 20
Level 20 → [Deserto]     → Level 30
Level 30 → [Ruinas]      → Level 40
Level 40 → [Vulcao]      → Level 50
Level 50 → [Abismo]      → Infinito
```

### Requisitos de Acesso

| Zona | Requisito |
|------|-----------|
| Floresta | Nenhum (inicial) |
| Cavernas | Level 10 + Quest "O Treant" |
| Deserto | Level 20 + Quest "Forja Ancestral" |
| Ruinas | Level 30 + Quest "Farao" |
| Vulcao | Level 40 + Quest "Imperador" |
| Abismo | Level 50 + Quest "Senhor do Fogo" |

## Sistema de Instancia

```typescript
interface ZoneInstance {
  zoneId: string;
  partyId: string;
  players: PlayerId[];
  maxPlayers: 4;

  // Seed para RNG consistente
  seed: number;

  // Estado persistente
  state: {
    mobsKilled: MobId[];
    chestsOpened: ChestId[];
    questProgress: Record<QuestId, number>;
  };

  // Scaling
  difficulty: {
    base: number;           // Baseado no level da zona
    partyBonus: number;     // +20% HP/dano por player extra
    playerLevelBonus: number; // Ajuste se overlevel
  };
}
```

### Party Scaling

| Players | Mob HP | Mob Damage | Loot |
|---------|--------|------------|------|
| 1 | 100% | 100% | 100% |
| 2 | 150% | 120% | 125% |
| 3 | 200% | 140% | 150% |
| 4 | 250% | 160% | 175% |
