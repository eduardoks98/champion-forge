# Lista de Habilidades (80+)

## Visao Geral

Sistema de 80-100 habilidades que jogadores podem desbloquear e equipar.
Cada personagem escolhe 4 habilidades: Q, W, E, R (ultimate).

## Como Desbloquear

### Metodos de Obtencao

| Metodo | Habilidades |
|--------|-------------|
| Level de conta | Tier 1 (basicas) |
| Missoes | Tier 2 (intermediarias) |
| Boss drops | Tier 3 (avancadas) |
| Dungeons | Ultimates |
| Arena tokens | Alternativas |

### Tabela de Unlock por Level

| Level | Habilidades Desbloqueadas |
|-------|---------------------------|
| 1 | 3 basicas (uma de cada cat) |
| 5 | +5 basicas |
| 10 | +5 tier 2 |
| 20 | +10 tier 2 |
| 30 | +10 tier 3 |
| 40 | +5 ultimates |
| 50 | Todas disponiveis via farm |
